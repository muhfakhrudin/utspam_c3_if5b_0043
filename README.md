# ðŸš— Aplikasi Rental Mobil - UTS Mobile Programming

Aplikasi penyewaan mobil berbasis Flutter yang terintegrasi dengan SQLite untuk manajemen data lokal. Aplikasi ini dibuat untuk memenuhi tugas Ujian Tengah Semester (UTS).

## ðŸ“± Fitur Utama
1.  **Registrasi & Login:** Validasi input dan penyimpanan akun ke database lokal.
2.  **Beranda (Home):** Menampilkan rekomendasi mobil, pencarian interaktif, dan navigasi modern.
3.  **Daftar Mobil:** Katalog lengkap mobil yang tersedia untuk disewa.
4.  **Formulir Sewa:** Input data penyewaan dengan fitur *Date Picker* dan kalkulasi harga otomatis.
5.  **Riwayat Transaksi:** Melihat status pesanan (Aktif/Dibatalkan) dan total biaya.
6.  **Edit & Batal:** Fitur untuk mengubah jadwal sewa atau membatalkan pesanan.
7.  **Profil Pengguna:** Menampilkan informasi akun yang sedang login.
